# AstroPing - Astrology Consultation Website

A modern, responsive astrology consultation website built with React and Vite. AstroPing connects users with professional astrologers for chat, call, and horoscope services.

## Features

- **Responsive Design**: Fully responsive layout that works on all devices
- **Modern UI/UX**: Clean, modern design with smooth animations and hover effects
- **Hero Section**: Eye-catching gradient background with call-to-action
- **Service Categories**: Chat, Call, Horoscope, and Store options
- **Astrologer Profiles**: Showcase of top online astrologers with ratings
- **Appointment Scheduling**: Easy booking system for various services
- **Daily Predictions**: Zodiac sign predictions and horoscope services
- **Free Services**: Panchang, Nakshatra, and Moon Sign calculations
- **Blog Section**: Educational content about astrology
- **Statistics**: Customer and astrologer count display

## Tech Stack

- **Frontend**: React 18.3.1
- **Build Tool**: Vite 5.4.2
- **Styling**: Pure CSS with modern features
- **Icons**: Unicode symbols and emojis
- **Images**: High-quality stock photos from Pexels
- **Fonts**: Inter font family from Google Fonts

## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm or yarn package manager

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd astro-ping
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
